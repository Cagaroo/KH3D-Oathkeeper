# Keyblades - Oathkeeper
Weapon Model replacement for Kingdom Hearts - Dream Drop Distance HD. 

A Keyblade imbued with light power, that will guard you from many hurts. Its powers help accelerate your growth. Replaces Knockout Punch for Sora and Riku.

### Installation 

This repository includes files for both 3DS and PC. See below for instructions to install for each platform.

#### PC

1. In OpenKH Mods Manager, under Mods, Install a new Mod
2. Enter this repository's name **Cagaroo/KH3D-Oblivion**
3. Wait for the file to complete its download
4. Ensure this mod is checked along with any other mods
5. Under Mod Loader, Build and Run

#### 3DS - Hardware Patching

1. Download the corresponding .rbin file from [GameBanana](https://gamebanana.com/members/submissions/mods/5286717).
2. In your Luma config menu, verify "enable game patching" is checked
3. Extract the zip and copy the "romfs" folder into luma/titles/
4. Replace for your region of the game:
    - US: 000400000008D300
    - PAL: 0004000000095500

#### 3DS - Citra Emulator

1. Download the corresponding .rbin file from [GameBanana](https://gamebanana.com/members/submissions/mods/5286717).
2. In the emulator's game list, right click your ROM and select "Open Mods location"
3. Extract the zip and copy the "romfs" folder into the Mods folder
